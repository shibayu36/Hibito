# Product Overview

Hibitoは**「今日のやる気を上げるためだけのTODOアプリ」**です。毎日0時（カスタマイズ可能）に全タスクが自動的に消去されるという独自のコンセプトを持つiOS/macOSアプリケーションです。

## Core Capabilities

- **1日限りのタスク管理**: 毎日決まった時刻にタスクを自動リセット
- **シンプルな操作**: タスクの追加、完了チェック、並び替え、削除
- **カスタマイズ可能なリセット時刻**: ユーザーが好きな時刻にリセットタイミングを設定可能
- **iCloud同期**: 複数デバイス間でのシームレスなタスク同期

## Target Use Cases

- **今日やることに集中したい人**: 明日以降のタスクが視界に入らず、「今日」に集中できる
- **完璧主義に疲れた人**: やり残したタスクは自動的にリセットされるので、精神的な負担が少ない
- **毎日をフレッシュに始めたい人**: 過去のタスクに引きずられず、毎朝ゼロからスタート

## Value Proposition

- **YAGNI原則の徹底**: 今必要な機能のみを提供（タスクの繰り返し、プロジェクト管理、タグなどは実装しない）
- **段階的な改善**: 完璧主義を避け、新機能から新しいパターンを適用する実用的なアプローチ
- **心理的なフレッシュスタート**: タスクが毎日リセットされることで、「今日は今日」という気持ちの切り替えをサポート
